01 — Принципы

Швейцарская
типографика

Международный типографический стиль, рождённый в Швейцарии в 1950-х, заложил математические основы, на которых сегодня строится каждый современный сайт.

01

Модульная сетка

Страница делится на колонки и поля, создавая невидимый каркас. Этот же принцип — основа CSS Grid Layout, где grid-template-columns задаёт те же колонки.

02

Визуальная иерархия

Размер, вес и положение элемента определяют его значимость. В CSS это выражается через font-size, font-weight, z-index и порядок в DOM.

03

Асимметричный баланс

Швейцарские дизайнеры отвергли симметрию ради динамичной композиции. CSS Grid с именованными областями позволяет создавать те же асимметричные макеты.

04

Математическая точность

Пропорции, золотое сечение, модульные отношения. В вебе это rem-единицы, калькуляции через calc() и пропорциональные отношения в clamp().

05

Гротескная типографика

Helvetica, Univers, Akzidenz-Grotesk — шрифты без засечек стали стандартом. Сегодня Inter, Roboto и SF Pro продолжают эту традицию в интерфейсах.

06

Негативное пространство

Пустота — активный элемент композиции. В CSS это margin, padding и gap. Швейцарцы знали: воздух на странице так же важен, как контент.

02 — Эволюция

Хронология
грид-систем

От карандашных разметок на бумаге до декларативных CSS-правил — каждый этап развивал идеи, заложенные швейцарскими дизайнерами.

1950-е
Швейцарская модульная сетка
Йозеф Мюллер-Брокманн и Эмиль Рудер систематизируют принципы модульной сетки. Книга «Grid Systems in Graphic Design» (1981) становится библией дизайнеров. Страница делится на колонки, поля и горизонтальные направляющие.
Демонстрация печатной сетки
1996
Табличная вёрстка
Первые веб-дизайнеры использовали HTML-таблицы для создания макетов. Невидимые таблицы имитировали колонки печатных полос. Это было костыльное, но работающее решение — дух сетки выжил в вебе.
Демонстрация табличной вёрстки
2000-е
Float и Clear
CSS-свойство float, созданное для обтекания текста вокруг изображений, стало основным инструментом создания колонок. clearfix-хаки, отрицательные margin — эпоха борьбы с потоком документа.
Демонстрация float-сетки
2012
Flexbox
CSS Flexible Box Layout революционизировал одномерные раскладки. Элементы стали выравниваться, распределяться и перестраиваться без хаков. Flexbox — это «умная» строка, которая адаптируется к контенту.
Демонстрация Flexbox
2017
CSS Grid Layout
Двумерная система, которая наконец воплотила мечты швейцарских дизайнеров в коде. Колонки, строки, именованные области, автоматическое размещение — всё, что рисовали карандашом на миллиметровке, стало декларативным CSS.
Демонстрация CSS Grid
2023+
Subgrid, Container Queries, Masonry
Новое поколение грид-технологий. Subgrid позволяет вложенным элементам наследовать родительскую сетку. Container Queries дают компонентную адаптивность. Masonry-раскладка — мечта каждого дизайнера.
03 — Сравнение

Печать vs Код

Одна и та же идея, разделённая 70 годами технологий.

Печатная вёрстка
Модульная сетка
Мюллера-Брокманна

Дизайнер рисует карандашом на кальке: поля, колонки, направляющие. Каждый элемент размещается по математическим правилам. Сетка невидима, но определяет всё.

CSS Grid Layout
Та же сетка,
другой язык

Вместо карандаша — CSS. Вместо кальки — браузерный рендерер. Но принципы те же: поля, колонки, пропорции. Код стал новым инструментом типографа.

/* Та же сетка в CSS */
.layout {
  display: grid;
  grid-template-columns:
    60px 1fr 1fr 1fr 60px;
  gap: 2px;
  padding: 2rem;
}

/* Поля = 60px, колонки = 1fr */
/* gap = швы между колонками */
